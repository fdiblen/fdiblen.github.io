## :watch: 14:35 (10 mins) - Create a Python package using a minimum profile

Install copier

``` { .sh .copy .select linenums="0" title="" }
python3 -m pip install --user pipx
python3 -m pipx ensurepath
pipx install copier
```

!!! info

    We will create a package called `baklava`

